Placez ici votre image de bannière, nommée exactement :

    banner-landing.jpg

Recommandations :
- Format paysage (ex. 1600x900 ou plus large), JPG ou PNG.
- L'image est automatiquement affichée en transparence (opacity 40%)
  avec un voile sombre dégradé par-dessus (voir landing.html), donc une
  image assez lumineuse/contrastée fonctionne mieux qu'une image déjà sombre.
- Si le fichier est absent, la page affiche simplement un dégradé de
  couleur de secours (aucune erreur, juste pas d'image).
